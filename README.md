# 成大資訊營隊網站 🍳

這是一個為 **2025 年成大資訊營隊** 設計的靜態網站，主打「**早餐主題風格**」。  
網站支援 **淺色／深色模式切換**，提供營隊資訊、課程內容與報名方式等。

- ☀️ **淺色模式**：使用溫暖的淡咖啡色、柔和橘黃色與米白色調，營造出早餐時光的氛圍  
- 🌙 **深色模式**：採用 Discord 風格配色，給你舒適的夜間瀏覽體驗
