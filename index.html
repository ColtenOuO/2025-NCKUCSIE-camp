交通資訊

開車：由台南交流道下高速公路，沿小東路直行至東寧路右轉，續行至長榮路左轉即可抵達成大。

火車：抵達台南火車站後，步行約10分鐘或搭乘計程車至成大光復校區。

公車：搭乘台南市公車至「大學路口」站下車，步行約5分鐘。

如何報名

報名方式：請填寫線上報名表單，報名截止日期為2025年6月30日。

繳費方式：報名確認後，請於7天內完成匯款，費用為新台幣8,000元，含課程、住宿與三餐。

退費政策：報名截止前可全額退費，活動開始前7天內退費50%，活動開始後恕不退費。

課程內容

必修Py (3hr)

  • 輸出、輸入
  • 資料型態&型別轉換
  • 算術運算 / round / abs
  • String format / f-string
  • 字串操作
  • 流程控制-判斷
  • 流程控制-迴圈
  • List
  • List Slice / String Slice
  • Dict
  • Function

必修C++ (3hr)

  • 輸出、輸入
  • 資料型態&型別轉換
  • 算術運算/運算子
  • 字串操作
  • 流程控制-判斷
  • 流程控制-迴圈
  • Function

資結/演算法++ (8hr)

  • vector
  • stack/queue/deque
  • set/map
  • priority_queue

爬蟲 (4hr)

  • 爬蟲概念
  • Html架構講解
  • 動態網站與靜態網站
  • 靜態網站爬蟲實作 (requests/bs4)
  • 動態網站爬蟲實作 (Selenium)

Discord bot (2hr)

  • 建立 Discord bot 帳號
  • 實作指令
  • 實作Event
  • 更多API查詢教學

電腦網路概論 (2hr)

  • 網路是什麼？為什麼我們需要他？
  • 有限網路與無線網路
  • 網路資料的交換 (封包介紹)
  • 網路協定簡述，為什麼我們需要協定
  • 網路分層介紹
  • IP 位置
  • DNS 簡述 (Domain Name Server)
  • 內網？外網？子網域？Port ?
  • 路由器 (Router) 與交換機 (Switch)
  • Mac 位置

Winform (2hr)

  • 工具箱介紹
  • 按鈕設計
  • 文字框與字串處理
  • 介面切換
  • MessageBox顯示

Arduino (2hr)

  • 電路與訊號基礎介紹 (電壓、電流、V=IR 等基礎觀念)
  • Arduino 語法解說
  • LED 電路實作
  • 按鈕 & LED 電路實作
  • 計時器與非同步閃燈實作
  • 序列傳輸通訊
